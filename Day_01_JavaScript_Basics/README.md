# Day 01 - JavaScript Basics 🚀

## Overview

This project contains my JavaScript fundamentals practice while learning Frontend Development.

The program demonstrates the use of basic JavaScript concepts such as variables, operators, conditional statements, logical operators, template literals, and simple problem-solving exercises.

---

## Files

### index.html

A simple HTML page that loads the JavaScript file.

### app.js

Contains JavaScript programs demonstrating various concepts and exercises.

---

## Topics Covered

### Console Output

* Using `console.log()`
* Displaying text and variables in the console

### Variables

* Declaring variables using `let`
* Storing numeric and string values

### Template Literals

* String interpolation using backticks (`` ` ``)
* Embedding expressions using `${}`

### Arithmetic Operators

* Addition
* Subtraction
* Multiplication
* Division

### Unary Operators

* Post Increment (`x++`)
* Pre Increment (`++x`)

### Comparison Operators

* Greater Than (`>`)
* Less Than (`<`)
* Greater Than or Equal To (`>=`)
* Less Than or Equal To (`<=`)
* Equal To (`==`)
* Not Equal To (`!=`)

### Conditional Statements

* `if`
* `else if`
* `else`

#### Programs Implemented

* Traffic Light Simulation
* Voting Eligibility Check
* T-Shirt Size and Price Calculator

### Logical Operators

* AND Operator (`&&`)

### String Problems

* Check whether a string is a "Good String"

### Number Problems

* Check whether two numbers have the same last digit
