# Day 05 - JavaScript Functions & Scope 🚀

## Overview

This project contains my practice and learning of JavaScript Functions, Function Arguments, Return Statements, Scope, Function Expressions, and Higher-Order Functions.

The programs are executed through an HTML file and the output can be viewed in the browser's Developer Console.

---

## Topics Covered

### 1. Basic Functions

* Creating and calling functions.
* Understanding function syntax and execution.

Examples:

* Print "Hello"
* Print custom messages
* Print numbers from 1 to 5

---

### 2. Function Practice Exercises

#### Print a Poem

A function that prints a simple poem line by line.

#### Dice Roll Simulator

Generates a random dice value between 1 and 6 using:

* `Math.random()`
* `Math.floor()`

---

### 3. Functions with Arguments

Passing parameters to functions and using them dynamically.

Example:

* Print a person's name and age.

---

### 4. Average of Three Numbers

A function that:

* Accepts three numbers as arguments.
* Calculates and displays their average.

---

### 5. Multiplication Table Generator

A function that:

* Accepts a number.
* Prints its multiplication table from 1 to 10.

---

### 6. Return Statements

Understanding how functions return values.

Examples:

* Sum of two numbers.
* Sum of numbers from 1 to N.

---

### 7. Array and String Operations

#### Concatenation of Array Elements

A function that:

* Accepts an array of strings.
* Returns a single concatenated string.

Example:

Input:
["abc", "def", "ghi"]

Output:
"abcdefghi"

---

### 8. Scope in JavaScript

#### Global Scope

Variables accessible throughout the program.

#### Function Scope

Variables declared inside a function.

#### Lexical Scope

Inner functions accessing variables from parent functions.

Example:

* Greeting message demonstration using nested functions.

---

### 9. Function Expressions

Understanding anonymous functions assigned to variables.

Example:

* Store a function inside a variable.
* Execute it like a normal function.

---

### 10. Higher-Order Functions

Functions that:

* Accept other functions as arguments.
* Execute functions multiple times.

Example:

* Greeting function executed repeatedly.

---

### 11. Factory Functions

A function that returns another function based on user requirements.

Example:

* Odd Number Checker
* Even Number Checker

Concepts used:

* Returning functions
* Dynamic function generation
* Closures
