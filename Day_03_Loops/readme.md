# Day 03 - JavaScript Loops, Arrays & Mini Projects 🚀

## Overview

This project contains my JavaScript practice exercises focused on loops, arrays, conditional statements, and basic problem-solving.

The goal of this module is to strengthen my understanding of JavaScript fundamentals through hands-on coding exercises and mini projects.

---

## Topics Covered

### 1. Array Element Deletion

* Remove a specific element from an array using `splice()`.
* Display the original and modified arrays.

### 2. Count Digits in a Number

* Accept user input using `prompt()`.
* Validate numeric input using `isNaN()`.
* Calculate the number of digits entered.

### 3. Sum of Digits

* Extract individual digits using the modulus operator (`%`).
* Calculate the sum of all digits in a number.

### 4. Factorial of a Number

* Use a `for` loop to calculate the factorial of a given number.

### 5. Todo List Application

A simple command-line style Todo application supporting:

* Add Tasks
* List Tasks
* Delete Tasks
* Quit Application

Concepts used:

* Arrays
* Loops
* Conditional Statements
* User Input Handling

### 6. Odd and Even Number Programs

* Print odd numbers from 1 to 20.
* Print even numbers from 2 to 20.

### 7. Break Statement

* Demonstrates how the `break` statement exits a loop when a condition is met.

### 8. Array Traversal

* Iterate through arrays using:

  * Traditional `for` loops
  * `for...of` loops

### 9. Nested Arrays

* Traverse nested arrays using nested loops.
* Access multidimensional array elements.
