# JavaScript Advanced Functions and Timing Methods

## Overview

This file contains practice examples of important JavaScript concepts including object getters, error handling, arrow functions, implicit returns, and asynchronous functions such as `setTimeout()` and `setInterval()`.

## Topics Covered

### 1. Object Getters

* Created a `student` object with properties:

  * name
  * age
  * eng
  * math
  * phy
* Implemented a getter method `avg` to calculate and display the average marks.
* Used the `this` keyword to access object properties.

### 2. Try-Catch Error Handling

* Demonstrated the use of `try...catch` blocks.
* Prevented program crashes caused by undefined variables.
* Learned how JavaScript handles runtime errors gracefully.

### 3. Arrow Functions

Implemented multiple arrow functions:

* Sum of two numbers
* Power calculation using exponent operator (`**`)
* Cube of a number
* Greeting function

### 4. Implicit Return

* Created a concise arrow function using implicit return syntax.
* Demonstrated cleaner and shorter function definitions.

### 5. setTimeout()

* Used `setTimeout()` to execute code after a specified delay.
* Learned asynchronous execution behavior in JavaScript.

### 6. setInterval()

* Used `setInterval()` to repeatedly execute code at fixed intervals.
* Stored the interval ID for future control and management.

## Key Concepts Learned

* Object Getters
* `this` Keyword
* Error Handling with Try-Catch
* Arrow Functions
* Implicit Return
* Asynchronous JavaScript
* setTimeout()
* setInterval()
* Callback Functions

## Learning Outcome

By completing this practice file, I gained hands-on experience with modern JavaScript function syntax, object methods, error handling mechanisms, and asynchronous programming concepts that form the foundation for DOM Manipulation, Events, APIs, React, and Node.js development.
